## 如何更新文档

当项目发生以下变更时，请同步更新对应文档：

1. **新增/删除依赖** → 更新 `01-项目概览.md` 技术栈部分
2. **新增目录/文件** → 更新 `02-目录结构.md`
3. **核心逻辑变更** → 更新 `03-核心模块.md`
4. **新增组件** → 更新 `04-组件库.md`
5. **新增 Hooks** → 更新 `05-Hooks 库.md`
6. **设计系统变更** → 更新 `06-设计系统.md`
7. **后端 API 变更** → 更新 `07-后端架构.md`
8. **规范变更** → 更新 `08-开发规范.md`

### 使用 AI 助手更新

你可以让 Claude 帮助更新文档：

```
请根据最新的代码变更，更新 docs/architecture/ 目录下的架构文档。
```
