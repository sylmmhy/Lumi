## 3.反思调整动态难度 （Leo）

根据记忆和你的“wish”之间产生的差距。

然后我们前面有一些理论上的东西。帮他把难度动态降低。

然后重新设置第二天的计划。

比如说我第一天，我希望11点钟睡觉，但事实上我平时都是3点睡觉。这时候，AI建议你不如2点睡吧，但我最后还是3点钟睡觉。这时候，AI就应该降低它的难度，变成你今天2点50睡觉吧，而不是设定到2点钟。 

**功能描述**：防止“连续两次错过” (Never Miss Twice) 。

- **逻辑**：
    - IF `Task_Status == Failed` for 2 consecutive days:
    - **Action**：主动建议降级。“看起来‘阅读30分钟’太难了。为了保护你的连续性，我们将明天的目标降级为‘阅读2分钟’。同意吗？”

### 难度伸缩与金发女孩准则

Planner 必须具备动态调整任务难度的能力。依据**金发女孩准则（The Goldilocks Rule）**，任务必须处于“难度适中”的区间才能维持动力 。

- **初始状态：** 对于新手，Planner 强制执行**两分钟规则（Two-Minute Rule）** 。它不会规划“阅读30分钟”，而是规划“阅读一页”。
- **逻辑：** 优化启动延迟（Start Latency）。就像优化网页的首字节时间（TTFB）一样，Planner 的首要目标是最小化行为启动的阻力。
